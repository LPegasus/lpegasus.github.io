useEffect

useEffect 是第二最常用的 hooks

useEffect 在浏览器环境才会执行，在服务端环境不会执行

通常 useEffect 用来处理 react 的副作用，常见的如：

  • 注册监听器
  • 创建计时器
  • 做 DOM 操作
  • I/O 操作
